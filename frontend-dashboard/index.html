📡 How This Dashboard Works

Every hour, an AWS Lambda function fetches live river data from Germany's PEGELONLINE API and saves a latest_status.json file to an S3 bucket. This dashboard uses fetch() to read that JSON and display the results.

For this to work, the S3 bucket needs CORS (Cross-Origin Resource Sharing) enabled — otherwise the browser blocks the request because the dashboard's domain differs from the S3 domain.
